## 功能

爬取网站上小说章节的内容，并以txt形式保存在本地

程序运行需要的参数如下：

| 参数名 | 参数值          | 样例                              |
|-----|--------------|---------------------------------|
| -f  | 保存在本地的文件名    | 斗破苍穹                            |
| -u  | 小说章节列表的url链接 | https://www.52bqg.org/book_361/ |

使用样例如下：
```shell
novel_crawler.exe -f 斗破苍穹 -u https://www.52bqg.org/book_361/
```
